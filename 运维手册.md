### 地震滑坡评估系统运维手册

- 环境（目前环境）
  windows 500G SSD + 32G内存
  QGIS 3.40.15
  增加的依赖包：

```
    Django==4.2.26
    Pillow==11.1.0
    GDAL==3.12.1
    pyshp==2.3.1
    rasterio==1.5.0
    requests==2.32.4
    concurrent-log-handler==0.9.29
    python-docx==1.1.2
    pykrige==1.7.3
    python-docx==1.1.2
    pip install docxtpl
    pip install jenkspy
    pip install mysqlclient==2.2.1 --only-binary=:all: -i https://pypi.doubanio.com/simple/
    pip install numpy==1.26.4 --force-reinstall -i https://pypi.tuna.tsinghua.edu.cn/simple
```
- Mysql 8.0

- 架构
   Django + MySQL + LayUI + JavaScript + HTML +   QGIS 3.40.15
- 系统启动
```
  进入 QGIS 3.40.15(快捷方式) -》OSGeo4W Shell执行
  "D:/App/dev/QGIS3.40.15/bin/python-qgis-ltr.bat" manage.py runserver 0.0.0.0:8000
  注意：D:/App/dev/QGIS3.40.15/ 路径需修改
```
- 替换底图
  ```angular2html
  进图 源码目录--》warehouse_manage目录-》settings.py
```

-- Mysql 启动
```
   次电脑--》右键--》管理-->服务 -》找到MySQL服务，右键启动
```

-- 数据库表管理
  ```
    mysql安装路径：mysql  -u root -p   回车
    use warehouse_manage;  # 切换到数据库
    show tables;  # 显示表
    desc table_name;  # 显示表结构
    select * from table_name;  # 显示表数据
    delete from table_name where id=1;  # 删除数据
    update table_name set field_name='value' where id=1;  # 更新数据
    insert into table_name (field1, field2) values ('value1', 'value2');  # 插入数据
    exit;  # 退出mysql
  ```

-- 系统问题终极解决方案 重启大法
```
   进入 QGIS 3.40.15(快捷方式) -》OSGeo4W Shell执行 ctrl+ c
   参考系统启动 重启
```
